# 这是什么
这是【东半球最好用的图床外链】，除了拥有酷酷的外观，更多的是细节处理
# 有哪些功能
- 同页面多文件上传
- 支持拖拽上传
- 图片支持[img]标签直接复制
- 支持各种缩略图/旋转变换组合
- 拥有配套chrome扩展，已发布至chrome store
- 等等
